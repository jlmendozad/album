# Panini Matcher

Webapp estática para controlar álbum, escanear correspondencias Panini y generar cruces de intercambio.

## Uso local

```bash
npm install
npm run dev
```

Luego abre:

```text
http://localhost:3000
```

## Variables de entorno

Crea `.env.local` a partir de `.env.example`:

```bash
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

## Despliegue

El proyecto está preparado para Vercel como app Next.js.

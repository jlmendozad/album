# Panini Matcher

Webapp estática para controlar álbum, escanear correspondencias Panini y generar cruces de intercambio.

## Uso local

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

Luego abre:

```text
http://127.0.0.1:8000/index.html
```

## Despliegue

El proyecto está preparado para Vercel como sitio estático.
